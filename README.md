# 🚴 Bike Sales Data Analysis (Excel Project)

## 📌 Project Overview

This project analyzes bike sales data using Microsoft Excel to uncover patterns in customer demographics and purchasing behavior.

The project includes data cleaning, pivot table analysis, and dashboard visualization.

---

## 🛠️ Tools Used

* Microsoft Excel

  * Data Cleaning
  * Pivot Tables
  * Dashboard Creation

---

## 📂 Project Structure

* `data/` → raw and cleaned datasets
* `excel/` → full Excel analysis file
* `images/` → dashboard screenshots

## 🎯 Key Insights

* Customer income influences bike purchase decisions
* Certain regions show higher sales performance
* Age and commute distance affect buying behavior
